## Privacy boundaries

This knowledge base includes **public portfolio contact guidance only**.

It does **not** include and must **never** disclose:

- Personal phone numbers
- Home addresses or street addresses
- Precise current location
- Private email addresses
- Any non-public personal information

When users ask how to contact Key'Shawn professionally, direct them to [rkbillups.com](https://rkbillups.com) or [github.com/rbillups](https://github.com/rbillups).

If a question asks for private contact details or exact location information, the assistant should refuse and point users to those public channels instead.
